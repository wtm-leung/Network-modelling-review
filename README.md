
Data and `R` code to support Leung *et al.*, (2023). Simulating contact networks for livestock disease epidemiology: a systematic review. *J. R. Soc. Interface.*

--------------------------------------------------------------------------------

Files:

  **figures.R:** `R` script for generating figures included in the publication.
  
  **data:** folder containing a folder containing the data extraction table from the systematic review which includes a) models.csv - details of each distinct model included; b) papers_years.csv - a list of the years each paper was published and the model framework used.
  
  **figures:** a folder for saving the figures generated by the `R` script.

The analysis was performed using R version 4.2.0.
