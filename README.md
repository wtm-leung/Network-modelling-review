--------------------------------------------------------------------------------
This systematic review presents an overview of empirically informed, model-based approaches of network generation and inference that have been applied to simulate
networks of contacts between livestock populations.

--------------------------------------------------------------------------------

Files and folders:

  **figures.R:** `R` script for generating figures included in the paper.
  
  **data:** folder containing a folder containing the data extraction table from the systematic review which includes a) models.csv - details of each distinct model included; b) papers_years.csv - a list of the years each paper was published and the model framework used.
  
  **figures:** a folder for saving the figures generated by the `R` script.

The analysis was performed using R version 4.2.0 (2022-04-22 ucrt) - "Vigorous Calisthenics"
